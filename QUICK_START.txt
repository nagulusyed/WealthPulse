🐛 TROUBLESHOOTING:
═══════════════════════════════════════════════════════════════════════════

Badge not showing?
→ Check initNotificationsAndSettings() is called in initApp()
→ Verify HTML has id="notif-badge-sidebar"
→ Run updateNotificationBadge() in console

Settings not persisting?
→ Check browser is not in private mode
→ Verify localStorage is enabled
→ Check console for storage errors

Notifications not appearing?
→ Ensure checkBudgetAlerts() is in transaction save
→ Verify budget is set for category
→ Check notification settings are enabled

Theme not switching?
→ Verify applyTheme() called in init
→ Check CSS variables are defined
→ Ensure .dark-theme class on body
