TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

❌ Problem: "Cannot read property 'style' of null"
✅ Solution: Check all IDs match (notif-list, settings-container, etc.)

❌ Problem: Notifications view is blank
✅ Solution: Verify renderNotificationsList() function was added from addon.js

❌ Problem: Settings not saving
✅ Solution: Check saveSettings() function exists and localStorage works

❌ Problem: Badge not showing
✅ Solution: Verify updateNotificationBadge() called in initNotificationsAndSettings()

❌ Problem: Theme not switching
✅ Solution: Check applyTheme() function and .dark-theme class in CSS

❌ Problem: Budget alerts not appearing
✅ Solution: Verify checkBudgetAlerts() added to saveTxn() function
