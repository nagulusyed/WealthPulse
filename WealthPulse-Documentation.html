📋 Project Overview

WealthPulse is a personal finance management application that helps users track income and expenses, visualize spending patterns, and manage monthly budgets. It runs as both a web application and a native Android app.

Key Highlights

📊 Dashboard

Real-time overview with balance, income, expenses, savings rate, charts, and trends.

💸 Transaction Management

Add, edit, delete, search, and filter income and expense transactions.

🎯 Budget Tracking

Set monthly spending limits per category with color-coded progress bars.

📱 Cross-Platform

Runs as a web app in any browser and as a native Android app via Capacitor.

💾 Local Storage

All data stays on your device. No server, no cloud, no accounts required.

📥 Export / Import

Backup and restore your financial data as JSON files.

📊 Dashboard — Empty State

When the app first loads, the dashboard shows zero balances and empty charts, ready for the user to start adding transactions.

Figure 1: Dashboard empty state with summary cards showing ₹0 and empty charts

Dashboard Components

  • Summary Cards — Net Balance, Income, Expenses, and Savings Rate
  • Spending by Category — Doughnut chart (appears when expenses are added)
  • Income vs Expenses — 6-month trend bar chart
  • Recent Transactions — Quick view of the last 5 entries
  • Month Navigation — Browse data across different months

➕ Add Transaction Modal

Users can add transactions from any screen using the floating action button (purple "+" at bottom-right) or the "Add Transaction" button on the Transactions page.

Figure 2: Add Transaction modal with type toggle, description, amount, category, and date fields

📊 Dashboard — With Data

After adding transactions, the dashboard updates in real-time with financial summaries, charts, and trends.

Figure 3: Populated dashboard showing balance, income, expenses, spending chart, and trend chart

Live Calculations

  • Net Balance = Total Income − Total Expenses
  • Savings Rate = (Income − Expenses) / Income × 100%
  • Month-over-Month Trend — Compares current month to previous month
  • Category Breakdown — Doughnut chart showing expense distribution

💸 Transactions List

The Transactions view shows all recorded income and expenses with search and filter functionality.

Figure 4: Full transaction list with category icons, amounts, and dates

Features

  • Filter Tabs — View All, Income only, or Expenses only
  • Search — Real-time text search across descriptions and categories
  • Click to Edit — Tap any transaction to modify or delete it
  • Color Coding — Green for income (+), Red for expenses (−)
  • Category Icons — Emoji-based icons with matching colors

🏗️ Architecture & Data Storage

Application Architecture

The app follows an IIFE (Immediately Invoked Function Expression) pattern with no framework dependencies. State changes trigger explicit re-renders.

ComponentFileSize
HTML Structureindex.html16 KB
CSS Design Systemstyles.css21 KB
Application Logicapp.js19 KB

Data Storage Format

All data is stored as JSON strings in the browser's localStorage (or the Android app's private WebView storage).

KeyFormatContains
wp_transactionsJSON ArrayAll income and expense records
wp_budgetsJSON ObjectBudget limits keyed by category ID

Data Persistence

EventData Survives?
Close browser / app✅ Yes
Restart device✅ Yes
App update✅ Yes
Clear browser data❌ No
Uninstall app❌ No

Export / Import

Users can export all data as a timestamped JSON backup file (wealthpulse-backup-YYYY-MM-DD.json) and import it back to restore data. Files are validated before import.

📱 Android App

The web app is wrapped as a native Android application using Capacitor by Ionic. The same HTML/CSS/JS code runs inside a native Android WebView.
